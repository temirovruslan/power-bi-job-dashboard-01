## Overview

This **Power BI dashboard** shows what is happening in the **data job market in 2024**.  
You can easily explore **job demand, salaries, locations, and required skills** in one place.

**Dashboard file:** [`Data_Jobs_Dashboard.pbix`](Data_Jobs_Dashboard.pbix)

---

## What I Did

- Cleaned and prepared the data using **Power Query**
- Created **measures and KPIs** (job count, median salary, skills per job)
- Built clear **charts and maps**
- Added **filters, buttons, and drill-through** for easy navigation
- Designed a **simple and readable dashboard**
